# RVC-UI - 实时语音转换界面

基于 gui_v1.py 的 RVC 实时语音转换 GUI 界面，使用 Vue.js + Tauri 技术栈实现。

## 功能特性

- **模型加载**：支持选择 .pth 模型文件和 .index 索引文件
- **音频设备管理**：支持多种音频 API（WASAPI、DirectSound、MME）
- **实时参数调节**：响应阈值、音调、性别因子等参数实时调节
- **性能优化**：支持多进程、降噪、相位声码器等性能选项
- **现代化界面**：响应式设计，支持深色模式

## 界面功能

### 1. 模型加载区域
- **选择 .pth 文件**：选择训练好的 RVC 模型文件
- **选择 .index 文件**：选择对应的索引文件

### 2. 音频设备设置
- **设备类型**：选择音频 API（WASAPI/DirectSound/MME）
- **独占 WASAPI 设备**：启用 WASAPI 独占模式
- **输入/输出设备**：选择音频输入和输出设备
- **采样率设置**：使用模型采样率或设备采样率
- **重载设备列表**：刷新音频设备列表

### 3. 常规设置
- **响应阈值**：-60 到 0 dB，控制语音检测灵敏度
- **音调设置**：-16 到 16，调整音调高低
- **性别因子/声线粗细**：-2 到 2，调整声线特征
- **Index Rate**：0 到 1，索引混合比例
- **响度因子**：0 到 1，响度混合比例
- **音高算法**：支持 pm、harvest、crepe、rmvpe、fcpe

### 4. 性能设置
- **采样长度**：0.02 到 1.5 秒，控制处理块大小
- **harvest 进程数**：1 到 8，多进程并行处理
- **淡入淡出长度**：0.01 到 0.15 秒，平滑过渡
- **额外推理时长**：0.05 到 5 秒，推理缓冲时间
- **输入/输出降噪**：启用音频降噪处理
- **启用相位声码器**：改善音质

### 5. 控制区域
- **开始/停止音频转换**：控制实时转换开关
- **功能选择**：输入监听或输出变声模式
- **状态显示**：算法延迟和推理时间监控

## 技术架构

## Tauri 命令 API

### 设备管理
- `get_audio_devices()` - 获取音频设备列表
- `reload_devices()` - 重载设备列表
- `update_device_config(...)` - 更新设备配置

### 文件选择
- `select_pth_file()` - 选择 PTH 模型文件
- `select_index_file()` - 选择 Index 索引文件

### 语音转换控制
- `start_voice_conversion(config)` - 开始语音转换
- `stop_voice_conversion()` - 停止语音转换
- `update_parameter(param, value)` - 更新实时参数

## 使用说明

1. **启动应用**：运行 `npm run tauri dev` 启动开发版本
2. **选择模型**：点击"浏览"按钮选择 .pth 和 .index 文件
3. **配置设备**：选择合适的音频输入输出设备
4. **调节参数**：根据需要调整各项参数
5. **开始转换**：点击"开始音频转换"按钮
6. **实时调节**：在转换过程中可以实时调节参数

## 注意事项

- 当前版本仅实现了界面和事件处理，实际的语音转换功能需要集成 rvc-core
- 所有文件路径和设备信息目前为模拟数据
- 建议在正式使用前先测试音频设备配置
